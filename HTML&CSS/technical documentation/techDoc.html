Introduction

Web development involves creating websites and web applications for the internet. It primarily uses three core technologies: HTML, CSS, and JavaScript. Each of these languages serves a unique purpose and together they create dynamic and interactive web pages.

Basic Structure

Understanding the roles of HTML, CSS, and JavaScript is essential for web development. HTML structures the content, CSS styles it, and JavaScript makes it interactive. Together, they form the foundation of web development, enabling the creation of modern, dynamic websites and applications.

HTML provides the basic structure of the page.

CSS styles the HTML elements, controlling the layout and visual appearance.

JavaScript adds dynamic behavior and interactivity to the page.

HTML

HTML (HyperTexT Markup Language) is the standard markup language used to create the structure of web pages. It describes the content and layout of a page using elements and tags.

Elements are the building blocks of HTML. Tags indicate the beginning and end of an element in the following structure:

<opening tag>content</closing tag>.

Some elements such as images and anchors are self closing and will not have content or a closing tag. All of these elements functionality can be included in an opening tag utilizing attributes.

Attributes can be included within the tags to provide additional information about elements. For example an element can include a class attribute like so:

<p class="value">content</p>

Common Elements

  • Body: <body>
  • container for all of a page's content
  • HTML: <html>
  • all html files are within an over-arching html tag, this is the basic tag that defines an html document.
  • Headings: <h1> to <h6>
  • create six levels of document headings ranging from small to large to breakup the page
  • Paragraphs: <p>
  • basic method of including text
  • Anchor: <a>
  • creates hyperlinks
  • Images: <img>
  • include images
  • Lists: <ul>, <ol>, <li>
  • ul (unordered list) and ol(ordered list) include li (list items) to create organized lists of information

CSS

CSS (Cascading Style Sheets) is used to style and layout HTML elements. It controls the visual presentation of web pages, including colors, fonts, and layouts.

Selectors target specific HTML elements to apply styles to them. Selectors take the form of:

target{ property: value; }

common targets are an element, a class attribute identified by .class, an id attribute identified by #id but there are additional methods to target different parts of an html document.

Properties define the style to be applied to a target in the html document. Some common examples are changing colors, qualities of the font, the location of the element on the document through widht, height, position, margins, and padding, as well as the structure for how it is displayed.

Values are the setting for a property. For example a color property could be set to red, or a font-size property could be set to 16px.

JavaScript

JavaScript is a programming language that adds interactivity and dynamic behavior to web pages. It can manipulate HTML and CSS, handle events, and perform calculations.

Variables store data values.

var x = 10;

The scope of a variable can be global by declaring it outside of any function but when declared within a function it is local to that function.

Constants are created with the const keyword and cannot change value.

const PI = 3.14;

JavaScript has six primite data types

  • Boolean
  • null
  • undefined
  • Number
  • String
  • Symbol (unique and immutable instances)

and Object in addition to these

if...else statements come in the form of

if (condition_1) {

statement_1;

} else if (condition_2) {

statement_2;

} else {

statement_3;

}

for loops are written as follows:

for (initialExpression; condition; incrementExpression) {statement}

while statements are written as :

while (condition) statement

Functions are reuseable snippets of code that have a declaration using a keyword followed by the name of a function, a list of arguments to the functions, and the statements that define the function.

function name(argument1, argument2...) { statement... return statement; }
